#  2. Add Two Numbers 两数相加
**<font color=red>难度: Medium</font>**

## 刷题内容

> 原题连接

* https://leetcode.com/problems/add-two-numbers/description/

> 内容描述

```
给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807
```

## 解题方案

> 方法一：直接构造成列表，然后转置，相加得到结果列表，再用结构列表构造节点列表

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution(object):
    def addTwoNumbers(self, l1, l2):
        """
        :type l1: ListNode
        :type l2: ListNode
        :rtype: ListNode
        """

        list1, list2 = [l1.val], [l2.val]  # 将其转换为 列表
        while l1.next:
            list1.append(l1.next.val)  # [2, 4, 3]
            l1 = l1.next
        while l2.next:
            list2.append(l2.next.val)  # [5, 6, 4]
            l2 = l2.next

        num1 = ''.join([str(i) for i in list1[::-1]])  # 转置后再join成 '342'
        num2 = ''.join([str(i) for i in list2[::-1]])  # 转置后再join成 '465'

        tmp = str(int(num1) + int(num2))[::-1]  # 相加后再转置成 '708'

        # 构造listNode
        res = ListNode(int(tmp[0]))
        res_tmp = res  # 临时节点用于存储下一个节点

        for i in range(1, len(tmp)):  # 要从 下标1开始
            res_tmp.next = ListNode(int(tmp[i]))
            res_tmp = res_tmp.next
        return res


s = Solution()

l1 = ListNode(2)
l1.next = ListNode(4)
l1.next.next = ListNode(3)

l2 = ListNode(5)
l2.next = ListNode(6)
l2.next.next = ListNode(4)

res = s.addTwoNumbers(l1, l2)

while (res.next):
    print(res.val)
    res = res.next

print(res.val)

```



> 方法二：使用递归，每次算一位的相加
******- 时间复杂度: O(N)******- 空间复杂度: O(1)******


```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def addTwoNumbers(self, l1, l2):
        """
        :type l1: ListNode
        :type l2: ListNode
        :rtype: ListNode
        """
        if not l1 and not l2:
            return None  # 递归 如果两个节点都不存在就return
        elif not (l1 and l2):
            return l1 or l2  # 如果有一个为空，就返回 那个不为空的
        else:
            if l1.val + l2.val < 10:
                l3 = ListNode(l1.val + l2.val)
                l3.next = self.addTwoNumbers(l1.next, l2.next)
            else:
                # 大于10的, 当前值 - 10, next的值里加 1 (随便第一个值+1 或者 第二个+1 都可以)
                l3 = ListNode(l1.val + l2.val - 10)
                l3.next = self.addTwoNumbers(self.addTwoNumbers(l1.next, ListNode(1)), l2.next)
        return l3


s = Solution()

l1 = ListNode(2)
l1.next = ListNode(4)
l1.next.next = ListNode(3)

l2 = ListNode(5)
l2.next = ListNode(6)

res = s.addTwoNumbers(l1, l2)

while (res.next):
    print(res.val)
    res = res.next

print(res.val)

```
