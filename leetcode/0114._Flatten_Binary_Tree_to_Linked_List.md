# 114. Flatten Binary Tree to Linked List 二叉树展开为链表

**<font color=red>难度: Medium</font>**

## 刷题内容

> 原题链接

* https://leetcode-cn.com/problems/flatten-binary-tree-to-linked-list/

> 内容描述

```
给定一个二叉树，原地将它展开为链表。

例如，给定二叉树

    1
   / \
  2   5
 / \   \
3   4   6
将其展开为：

1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None
```

## 解题方案

> 方法一：后续遍历，每次遍历后将右指针指向正确的前置元素。时空复杂度O(n)，O(1)
>

```python
class Solution:
    def __init__(self):
        self.pre = None

    def flatten(self, root: TreeNode) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        if not root:
            return
        self.helper(root)

    def helper(self, node):
        if not node:
            return

        self.helper(node.right)
        self.helper(node.left)
        node.right = self.pre
        node.left = None
        self.pre = node
```

```python
class Solution:

    def flatten(self, root) -> None:
        """
        Do not return anything, modify root in-place instead.
        """

        if not root:
            return None

        if root.left:
            # 如果有左子树, 找到左子树的最右节点, 将root的右子树挂载上面
            pre = root.left
            while pre.right:
                pre = pre.right

            # 将root的右挂载左子树的最右边节点
            pre.right = root.right
            # 将整个左子树移到root的右边
            root.right = root.left
            # 更新左
            root.left = None

        # 尾递归继续调用
        self.flatten(root.right)
```



> 方法三：同方法二，将尾递归用迭代实现

```python
class Solution:
    def flatten(self, root) -> None:
        """
        Do not return anything, modify root in-place instead.
        """

        if not root:
            return None

        while root:
            if root.left:
                pre = root.left
                while pre.right:
                    pre = pre.right

                pre.right = root.right
                root.right = root.left
                root.left = None

            root = root.right
```
